# Mentoria

Mentoria is a premium advisory and education platform focused on sport, investment, and executive learning.

## Project structure

```text
index.html
login.html
README.md
assets/
  css/
    index.css
    login.css
  js/
    index.js
    login.js
    mobile-nav.js
  images/
```

## Pages

- `index.html` — landing page
- `login.html` — member login and registration page

## Notes for GitHub Pages

- Keep `index.html` in the root of the repository.
- Upload the full `assets` folder together with both HTML files.
- The mobile version now includes a visible **Member Login** button and hamburger menu.
- Internal links use relative paths, so the site works on GitHub Pages.

## Deploy

1. Upload all files to the repository root.
2. Commit changes.
3. In GitHub, go to **Settings → Pages**.
4. Publish from the `main` branch and the root folder.
